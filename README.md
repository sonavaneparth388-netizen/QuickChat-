# Family Alert SMS Web Application (Android Local)

A modern, mobile-friendly web dashboard designed to run entirely on an Android phone using Python Flask. It simulates broadcasting instant alerts ("I'm Safe", "Emergency", etc.) to predefined family members.

## 📂 Project Structure

Ensure your project directory looks exactly like this on your phone's storage:

```text
📁 FamilyAlert/
│
├── 📁 templates/
│   └── index.html
│
├── 📁 static/
│   ├── style.css
│   └── script.js
│
├── app.py
├── contacts.json
├── messages.py
├── sms_service.py
├── requirements.txt
└── README.md
